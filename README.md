# SQL Financial Analysis Case Study

## Project Overview
This project is a financial analysis case study completed during the **Data Analyst Bootcamp at TripleTen**. It demonstrates data exploration, integration, cleaning, and calculation of key financial metrics using **SQL** and a **Google Sheets/Excel dashboard**.

## Description
Performed 15 SQL queries organized in stages: schema exploration, data integration and cleaning, financial KPI calculation, and data validation. Since the original dataset is not available, all queries, instructions, and results are documented through **screenshots**. A dashboard was created in Google Sheets to visualize key financial indicators.

## Tools
- SQL (simulator provided by TripleTen)
- Google Sheets / Excel (for dashboard creation)
